cd <folder>

git clone - det kloner indholdet i github og smider det ind <url>

git status - fortæller nuværende status

git add . - Tilføjer alle filer i mappen

git commit -m "message" - Tilføjer en kommentar til filen

git push - Skubber filen ud til github

git pull

git config --global user.name <user>

git config --global user.email <email>
